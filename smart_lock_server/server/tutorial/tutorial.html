OpenDoorWithFace API Tutorial

This server exposes face registration and recognition routes. JSON routes accept base64 images. ESP32-S3-CAM should use the raw JPG routes because it can send the camera frame buffer directly.

Use your server LAN IP from ESP32, for example http://192.168.1.20:8000. Do not use localhost from the ESP32.

1. Load Model

Call this once before recognition or registration routes.

POST /model/init

{
  "det_size": [320, 320],
  "device": "cuda"
}

Response

{
  "message": "InsightFace model loaded successfully.",
  "status": "loaded"
}

2. Unload Model

Unload the model from memory.

DELETE /model/delete

Response

{
  "message": "InsightFace model unloaded successfully.",
  "status": "unloaded"
}

3. Upload JPG From ESP32-S3-CAM

Saves a raw JPG image to server/uploads. This route does not require the face model.

POST /image/upload

Headers:

Content-Type: image/jpeg

Body: raw JPG bytes.

Example Response

{
  "success": true,
  "message": "Image uploaded successfully.",
  "path": "uploads/20260508T120000Z_abc123.jpg",
  "width": 640,
  "height": 480,
  "bytes": 54321
}

4. Recognize Face From ESP32 JPG

Recognizes a face from raw JPG bytes sent by ESP32-S3-CAM. If a face is detected but not found in the database, the server saves the JPG and the receive time.

POST /face/recognize-jpg

Compatibility alias: /face/regconize-jpg

Headers:

Content-Type: image/jpeg

Body: raw JPG bytes.

Example Match Response

{
  "found": true,
  "metadata": {
    "faiss_id": 0,
    "user_name": "John Doe",
    "user_id": "2a4b7f3a-1cc0-4d74-a8be-3e78d2e5d6f2"
  },
  "distance": 0.6,
  "message": "Match found: John Doe"
}

Example Unknown Response

{
  "found": false,
  "metadata": null,
  "distance": 0.31,
  "message": "No matching face found in the database."
}

5. Get Saved Unknown Face Images

Returns saved unknown-face JPG images and the time each image was received by the server. Use number=-1 to return all saved images, or a positive number to return the newest images.

GET /face/unknown-jpg?number=-1

Example Response

{
  "count": 1,
  "images": [
    {
      "received_at": "2026-05-14T10:30:00+00:00",
      "path": "unknown_faces/20260514T103000Z_abc123.jpg",
      "image_base64": "/9j/4AAQSkZJRgABAQ...",
      "bytes": 54321
    }
  ]
}

6. Get Registered Users

Returns users from the face database with user names and saved registered face images. Use number=-1 to return all users, or a positive number to return the newest users.

GET /face/users?number=-1

Example Response

{
  "count": 1,
  "users": [
    {
      "faiss_id": 0,
      "user_name": "John Doe",
      "user_id": "2a4b7f3a-1cc0-4d74-a8be-3e78d2e5d6f2",
      "registered_at": "2026-05-14T10:35:00+00:00",
      "image_path": "registered_faces/20260514T103500Z_abc123.jpg",
      "image_base64": "/9j/4AAQSkZJRgABAQ...",
      "bytes": 54321
    }
  ]
}

7. Register Face From ESP32 JPG

Registers a face from raw JPG bytes. Send only the user name; the server generates user_id with UUID.

POST /face/register-jpg?user_name=John%20Doe

Headers:

Content-Type: image/jpeg

Body: raw JPG bytes.

Example Response

{
  "success": true,
  "faiss_id": 0,
  "user_id": "2a4b7f3a-1cc0-4d74-a8be-3e78d2e5d6f2",
  "message": "Successfully registered 'John Doe' with faiss_id=0."
}

8. Base64 JSON Register

Use this from web or mobile clients that already have a base64 image string.

POST /face/register

{
  "image_base64": "...",
  "user_name": "John Doe"
}

9. Base64 JSON Recognize

Use this from web or mobile clients that already have a base64 image string.

POST /face/recognize

{
  "image_base64": "..."
}

10. ESP32-S3-CAM Arduino Example

Send fb->buf directly as the POST body.

#include "WiFi.h"
#include "HTTPClient.h"
#include "esp_camera.h"

const char* serverUrl = "http://192.168.1.20:8000/face/recognize-jpg";

void sendPhoto() {
  camera_fb_t *fb = esp_camera_fb_get();
  if (!fb) {
    Serial.println("Camera capture failed");
    return;
  }

  HTTPClient http;
  http.begin(serverUrl);
  http.addHeader("Content-Type", "image/jpeg");

  int statusCode = http.POST(fb->buf, fb->len);
  Serial.printf("HTTP status: %d\n", statusCode);
  Serial.println(http.getString());

  http.end();
  esp_camera_fb_return(fb);
}
